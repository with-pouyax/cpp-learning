## 🛠️ **How to Use This Repository**
1. **Clone this repo** _(if you dare)_:  
   ```bash
   git clone https://github.com/with-pouyax/learn-cpp.git
   ```
2. **Enter the chaos**:
   ```bash
   cd learn-cpp
   ```
3. **Compile a random file & hope it runs**:
   ```bash
   g++ filename.cpp -o output
   ./output
   ```
